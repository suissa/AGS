# 02 — GitHub REST API Integration

Implemente a camada real de integração com GitHub REST API.

## Objetivo

Criar cliente REST para:

- criar repositório;
- criar issue;
- comentar issue;
- criar labels;
- criar milestones;
- criar/atualizar arquivos no repo;
- ler workflow runs;
- obter PRs e issues.

## Regras

- Use `fetch` nativo do Node.js 20+.
- Não use GitHub CLI.
- Não use Octokit nesta primeira versão.
- Token vem de `GITHUB_TOKEN`.
- Trate erros HTTP com mensagens claras.

## Arquivos esperados

```txt
src/github/github-rest-client.ts
src/github/github-errors.ts
src/github/types.ts
src/github/github-auth.ts
tests/github-rest-client.test.ts
```

## Métodos obrigatórios

```ts
createRepository(input)
createIssue(input)
createIssueComment(input)
createLabel(input)
createMilestone(input)
getIssue(input)
listIssues(input)
createOrUpdateFile(input)
listWorkflowRuns(input)
getPullRequest(input)
```

## Critérios de aceite

- Nenhum token hardcoded.
- Erros 401, 403, 404 e 422 tratados.
- Testes com mock de fetch.
